# Get-CosmosDbDatabaseResourcePath

## SYNOPSIS

Return the resource path for a database object.

## SYNTAX

```powershell
Get-CosmosDbDatabaseResourcePath [-Id] <String> [<CommonParameters>]
```

## DESCRIPTION

This cmdlet returns the resource identifier for a database
object.
